인디 게임
-직업 : 대검사
-성별 : 남성
-무기 구성 : 대검
전

-능력치 구성 및 수치 : 
체력 : 500hp
기력 : 300
물리공격력 : 65
물리 방어력 : 25
공격속도 : 3초에 기본 공격 두 번 가능
이동 속도 : 1초에 캐릭터 한명 거리
수영 속도 : 1초 캐릭터 반명 거리
점프사거리 : 캐릭터 한명 반 높이
심폐력 (마크의 공깃방울과 비슷한 개념) : 100 (숨참을시 1초에 20씩 닳음)
기본 공격 사거리 : 캐릭터 2개 정도 거리
방어 사거리 : 캐릭터 한개 정도 거리
지구력 (기력이 다는 속도) : 이동,점프,수영시 20씩 닳음 기본 공격,방어시 40씩 닳음
특공 : 0
특방 : 0
마공 : 0
마방 : 0
운 (이는 랜덤한 것이 아니며 캐릭터에 따라 인센티브가 있는 것 ex. 도박사 , 예언자 캐릭터 등) : 0 
체력 재생력 : 초당 0.2hp
기력 재생력 : 초당 5
몸무게 : 70 kg
정신력 (마법,특수 관령 cc기의 영향을 덜 받는 능력치) : 0
저항력 (물리 관련 cc기의 영항을 덜 받는 능력치) : 0
허기 : 100 (10초마다 5씩 닳음)
갈증 : 100 (10초마다 5씩 닳음)

-패시브 스킬 : 3개
패시브 1. 모든 기본공격과 공격 기술이 적에게 피격시 적을 뒤로 넉백시킴
패시브 2. 대검을 이용하는 기술의 피해가 5% 올라감
패시브 3. 5번째 중첩이 쌓인 상대는 물리 방어력이 감소함
-기술 트리 갯수 : 미정 (현재로썬 신경쓰지 않아도 되는 사항)
-인게임 내 기술 갯수 : 5개
기술 1. 3초동안 공격력 10% 강화 (자기 자신만) 재사용 대기 시간 : 11초 (기력소모 : 30)

기술 2. 전방으로 돌진. 유닛 충돌 무시하고 맞은 상대는 80의 데미지를 받음. 재사용 대기 시간 : 9초 (캐릭터 5명 거리를 돌진함) 기력 소모 : 70

기술 3. 검을 아래로 강하게 내리찍기. 상대를 쓰러짐 상태이상에 0.5초동안 걸리게 하고 40의 데미지를 받게 함. 재사용 대기시간 :  12초 (캐릭터 한명 정도 거리) 기력소모 : 50

기술 4. 대검을 사용하는 기술의 기력 소모가 -30%됨. (즉 원래 100의 기력을 소모한다면 70의 기력만 소모하고 스킬을 쓸 수 있음) 재사용 대기시간 : 7초 (자기 자신만) 기력소모 : 100

기술 5. 대검을 세로로 세워서 막기 대검에 닿는 물리 공격과 투사체 공격들을 1초 동안 무효화시킴. 재사용 대기시간 : 10초 (자기자신의 전방만) 기력 소모 : 80

게임내 역할군 : 데미지 딜러 , cc기 보유자 , 탱커 , 돌격병

아이템 슬롯 : 9개
